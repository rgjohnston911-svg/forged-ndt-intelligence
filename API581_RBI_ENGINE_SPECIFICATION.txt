================================================================================
ENGINE ARCHITECTURE
================================================================================

This Netlify function implements the complete quantitative RBI (Risk-Based
Inspection) methodology from API RP 581, 3rd Edition (2016). Every calculation
is defensible against refinery engineering audits and regulations.

Core Equation:
  RISK = PROBABILITY OF FAILURE (PoF) × CONSEQUENCE OF FAILURE (CoF)

The engine supports 8 primary actions:
  1. assess_risk      - Full RBI assessment (PoF + CoF + Risk Score)
  2. compute_pof      - Probability of failure calculation only
  3. compute_cof      - Consequence of failure calculation only
  4. compute_damage_factor - Damage mechanism quantification
  5. plan_inspection  - Inspection planning to achieve target risk
  6. get_risk_matrix  - 5×5 risk ranking matrix
  7. get_registry     - List available equipment types, fluids, mechanisms
  8. get_history      - Fetch prior assessment records

================================================================================
PROBABILITY OF FAILURE (PoF) - API 581 Section 5
================================================================================

GENERIC FAILURE FREQUENCIES (Table 5.1):
  Equipment Type              GFF (per year)
  ──────────────────────────────────────────
  Pipe                        3.06e-5 per meter
  Vessel                      7.56e-5 per item
  Tank Bottom                 6.26e-4 per item
  Heat Exchanger Tube         1.05e-3 per item
  Pump Casing                 2.41e-4 per item
  Fired Heater Tube           1.41e-4 per item
  Separator                   1.37e-4 per item
  Compressor Casing           1.05e-4 per item

DAMAGE FACTORS (Sections 5.3.1 - 5.3.6):

  1. THINNING DAMAGE FACTOR (Df_thin)
     ─────────────────────────────────
     Art = (Tnom - Tmm) / (Tnom - Tmin)

     where:
       Tnom = nominal wall thickness (mm)
       Tmm = current measured minimum thickness (mm)
       Tmin = minimum allowable thickness (mm)

     Df is looked up from Table 5.11/5.12 based on Art and inspection history:

     With 0 Category A inspections:
       Art=0.0→Df=1,    Art=0.2→Df=30,    Art=0.5→Df=5,100,   Art=0.8→Df=160,000
       Art=0.1→Df=1,    Art=0.3→Df=350,   Art=0.6→Df=18,000,  Art=0.9→Df=450,000
       Art=0.4→Df=1,500 Art=0.7→Df=55,000 Art=1.0→Df=1,200,000

     With 1 Category A inspection: divide by ~10
     With 2 Category A inspections: divide by ~50
     With 3 Category A inspections: divide by ~200

     Inspection Effectiveness (Section 5.3.1):
       Category A: POD ≥ 90% (Highly Effective)
       Category B: POD ≥ 70% (Usually Effective)
       Category C: POD ≥ 50% (Fairly Effective)
       Category D: POD ≥ 30% (Poorly Effective)
       Category E: POD < 30% (Ineffective)

  2. STRESS CORROSION CRACKING (Df_scc) - Section 5.3.3
     ──────────────────────────────────────────────────
     Depends on:
       - Susceptibility (high/medium/low/none)
       - Time in service (years)
       - Inspection history (number and negativity)

     Base calculation: Df_scc = A × exp(-B × t)
       High susceptibility:    A=3500,  B=0.15
       Medium susceptibility:  A=1000,  B=0.10
       Low susceptibility:     A=100,   B=0.05

     Reduction for clean inspection history: Df /= (1 + N×0.5) where N = inspections

  3. EXTERNAL CORROSION (Df_ext) - Section 5.3.4
     ────────────────────────────────────────────
     Factors:
       Coating condition: poor (×8), fair (×3), good (×0.5)
       Operating temp >80°C: multiply by (1 + (T-80)/100)
       Insulation present: ×2 (traps moisture), absent: ×1

     Df_ext = base × coating_factor × temperature_factor × insulation_factor

  4. HIGH-TEMPERATURE HYDROGEN ATTACK (Df_htha) - Section 5.3.5
     ────────────────────────────────────────────────────────────
     Uses simplified Nelson Curve zones:
       Zone 1 (Safe):     T < 400°F                           → Df = 1
       Zone 2 (Moderate): 400-600°F, pH2 < 1000 psi           → Df = 10-15
       Zone 3 (Risk):     > 800°F or high pressure            → Df = 100

     Material grade mitigation:
       1Cr1SrVo / 2Cr1SrVo: ×0.1
       1Cr0.5Mo:            ×0.3
       Others:              ×1.0

  5. BRITTLE FRACTURE (Df_brittle) - Section 5.3.6
     ───────────────────────────────────────────────
     Based on Charpy energy (MAT = MDMT + safety margin):
       Margin < 10°C:   Df = 20
       Margin < 20°C:   Df = 5
       Margin < 50°C:   Df = 2
       Margin ≥ 50°C:   Df = 1
       Below MAT:       Df = 50

     Thickness penalty (>1 mm): ×1.5

MANAGEMENT SYSTEMS FACTOR (FMS) - Section 5.4
──────────────────────────────────────────────
PSM Quality Score → FMS Multiplier:
  ≥ 4.5: 0.5  (Excellent program)
  ≥ 4.0: 0.6
  ≥ 3.5: 0.75
  ≥ 3.0: 1.0  (Baseline)
  ≥ 2.5: 1.25
  ≥ 2.0: 1.5
  ≥ 1.0: 1.75
  < 1.0: 2.0  (Poor/missing program)

PROBABILITY OF FAILURE CALCULATION
───────────────────────────────────
For piping: PoF = gff × length_m × max(Df_thin, Df_scc, Df_ext, Df_htha, Df_brittle) × FMS
For vessels: PoF = gff × max(Df_thin, Df_scc, Df_ext, Df_htha, Df_brittle) × FMS

per API 581 Section 5.2

================================================================================
CONSEQUENCE OF FAILURE (CoF) - API 581 Section 8
================================================================================

RELEASE RATE CALCULATION (Section 8.3.1)
─────────────────────────────────────────
For liquids:
  Qm = Cd × A × sqrt(2 × ρ × ΔP)

  where:
    Cd = discharge coefficient (≈0.7)
    A = hole area (m²)
    ρ = density (kg/m³)
    ΔP = operating pressure (Pa)

For gases:
  Qm = Cd × A × P × MW / sqrt(Z × R × T × MW × ρ)

RELEASE MASS CALCULATION
────────────────────────
Release_mass_kg = release_rate × (detection_time + isolation_time)

Default assumptions:
  Detection time: 60 seconds
  Isolation time: 300 seconds (5 minutes)
  Can be overridden per scenario

CONSEQUENCE AREA - FLAMMABLE (Table 8.1)
──────────────────────────────────────────
Consequence Area = a × mass^b

Fluid Categories:
  C1-C2 (Methane/Ethane)     a=8.669,  b=0.98  (jet fire auto-ignition)
  C3-C4 (Propane/Butane)     a=55.13,  b=0.95
  C5 (Pentane)               a=96.53,  b=0.92
  C6-C8 (Gasoline)           a=130.2,  b=0.90
  C9-C12 (Diesel)            a=103.4,  b=0.88
  C13-C16 (Gas Oil)          a=79.94,  b=0.85
  C17-C25 (Crude)            a=60.07,  b=0.82
  Hydrogen (H2)              a=16.47,  b=0.96  (delayed ignition → VCE)
  H2S                        Use toxic model
  Steam                      Use thermal model

Consequence radius = sqrt(area / π)

FINANCIAL CONSEQUENCE
─────────────────────
CoF_financial = Equipment_Value + (Production_Loss_per_day × Downtime_Days)
              + Environmental_Cost + Injury_Cost

Default assumptions:
  Downtime: 3 days
  Equipment value: $50,000 (override per asset)
  Production loss: $10,000/day (override per unit)
  Environmental: $100,000 (site-dependent)
  Injury: $0 (or calculated from population × injury severity)

POPULATION AT RISK
──────────────────
Estimated as: floor(consequence_radius_m × 10)
Used for societal risk calculation and regulatory compliance

RISK CALCULATION & RANKING
================================================================================

RISK SCORE
──────────
Risk = PoF × CoF_financial_usd

RISK MATRIX (Section 9.1 - 5×5)
────────────────────────────────
                   CoF LEVEL
                   ─────────────────────────────
PoF LEVEL   1          2          3          4          5
           (1)        (10)       (100)      (1000)     (10000)

1 (1e-5)    LOW        LOW        LOW        LOW        MED

Position on matrix drives inspection planning and acceptance decisions.

================================================================================
INSPECTION PLANNING - API 581 Section 7
================================================================================

ACTION: plan_inspection

Calculates optimal inspection intervals and techniques to achieve target risk.

INPUTS:
  - Current PoF (from assess_risk)
  - Target PoF (default: 1e-4)
  - Damage mechanism (thinning, scc, external_corrosion, htha, brittle_fracture)

OUTPUTS:
  - Recommended inspection technique (UT_Thickness, UT_Shear_Wave, etc.)
  - Required effectiveness rating (Category A/B/C/D/E)
  - Number of inspections to achieve target
  - Optimal inspection interval (years)
  - Next inspection date (ISO format)

INTERVAL CALCULATION
────────────────────
Number_of_inspections = ceil(log10(PoF_current / PoF_target))
Interval_years = min(5 years, max(1 year, 10 / Number_of_inspections))

Example:
  Current PoF:  1e-2
  Target PoF:   1e-4
  Ratio:        100
  Inspections:  ceil(log10(100)) = 2 inspections
  Interval:     10 / 2 = 5 years

================================================================================
DATABASE PERSISTENCE
================================================================================

Table: rbi_assessments

Columns:
  - assessment_id (UUID, auto-generated)
  - asset_id (text) - foreign key to asset registry
  - action (text) - assess_risk, compute_pof, etc.
  - equipment_type (text)
  - pof (numeric) - probability of failure
  - cof_usd (numeric) - consequence in USD
  - risk_level (text) - low/medium/high
  - input_data (jsonb) - full input parameters
  - result_data (jsonb) - full calculation results
  - created_at (timestamp) - assessment timestamp
  - updated_at (timestamp) - last update

Non-Fatal Database Operations:
  All Supabase write operations are wrapped in try/catch. Calculation
  results are returned even if persistence fails. This ensures the
  engine remains operational despite database unavailability.
