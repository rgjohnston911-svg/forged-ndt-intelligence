CALCULATED OUTPUTS
════════════════════════════════════════════════════════════════════════════════

1. MINIMUM REQUIRED THICKNESS (tmin)
   Per ASME Section VIII Div 1 or ASME B31.3 depending on component type
   - Cylinder: tmin = (P×R) / (S×E - 0.6×P)
   - Sphere: tmin = (P×R) / (2×S×E - 0.2×P)
   - Pipe/Elbow/Nozzle: Component-specific formulas

2. REMAINING THICKNESS RATIO (Rt) - Eq 5.9
   Ratio of remaining margin to allowable loss
   Rt = (tmm - FCA) / (tnom - FCA - tmin)

3. FLAW LENGTH PARAMETER (lambda) - Eq 5.5
   Characterizes flaw size relative to critical dimensions
   lambda = 1.285 × c / sqrt(D × tmin)

4. FOLIAS FACTOR (Mt) - Eq 5.11 / 5.12
   Critical stress concentration shape correction
   Two-regime algorithm: Mt switches equation at lambda² = 50
   (Low lambda: polynomial; High lambda: linear approximation)

5. REMAINING STRENGTH FACTOR (RSF) - Eq 5.13
   THE REGULATORY DECISION METRIC
   RSF = Rt / (1 - (1/Mt) × (1 - Rt))
   Controls: ACCEPTABLE | REDUCED_MAWP | REPAIR_REQUIRED

6. REDUCED MAWP (MAWPr) - Eq 5.14
   Safe operating pressure if RSF < allowable
   MAWPr = MAWP × (RSF / RSFa)

7. REMAINING LIFE - Eq 5.15 / 5.16
   Asset service life remaining
   Years = (tmm - tmin) / corrosion_rate
   Or with FCA: Years = (tmm - FCA - tmin) / corrosion_rate

8. ACCEPTANCE CRITERIA - API 579-1 Table 5.2
   RSF >= 0.9 (General Service) → ACCEPTABLE
   RSF >= 0.75 (High-Consequence) → ACCEPTABLE
   0.5 < RSF < threshold → REDUCED_MAWP
   RSF <= 0.5 → REPAIR_REQUIRED

9. FLAW INTERACTION ASSESSMENT
   Checks if multiple flaws require combined analysis
   Critical distance: 2 × sqrt(D × t)

10. CRITICAL THICKNESS PROFILE (CTP) ANALYSIS
    Grid-based thickness measurement extraction for Level 2 rigor
    Returns: min/max/avg thickness, range, point count

  MAWPr           DECIMAL (reduced max pressure)
  remaining_life_years  DECIMAL
  assessment_timestamp  TIMESTAMP
  created_at      TIMESTAMP DEFAULT NOW()

KNOWN LIMITATIONS & CONSTRAINTS
════════════════════════════════════════════════════════════════════════════════

LEVEL 2 (vs LEVEL 1 or LEVEL 3):
  - Level 2 assumes grid-based thickness measurements
  - Allows critical thickness profile extraction
  - More refined than Level 1 screening, less detailed than Level 3 FEA
  - Appropriate for routine refinery inspections

COMPONENT TYPES SUPPORTED:
  - Cylinder (most common)
  - Sphere
  - Pipe (B31.3 piping code)
  - Elbow (branch connection stress)
  - Nozzle (reduced strength factor)

COMPONENT TYPES NOT YET SUPPORTED:
  - Complex geometries (conical transitions, etc.)
  - Welds under flaw (requires partial failure analysis)
  - Multiple flaw combined assessment (future enhancement)

TEMPERATURE EFFECTS:
  - Material properties (S, E) must be pre-adjusted for operating temperature
  - Engine does not perform temperature de-rating
  - User responsible for providing temperature-adjusted allowable stress

MATERIAL ASSUMPTIONS:
  - Isotropic material properties (not valid for composites)
  - Standard code allowables apply (no creep regime)
  - Ductile material behavior (not brittle fracture assessment)
